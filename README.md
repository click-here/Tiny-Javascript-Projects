#### 4 Ways to Call a Function

##### Function form
`functionObject(arguments)`

##### Method form
`thisObject.methodName(arguments)`
`thisObject["methodName"](arguments)`

##### Constructor form
```javascript
new FunctionObject(arguments
```

##### Apply form
```javascript
functionObject.apply(thisObject,[arguments])
```
